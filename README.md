# Rei Foto
Aplicación web de galeria de fotos

### Dependencias

* NodeJS
* Express
* Nodemon
* Multer
* MongoDB
* Mongoose
* JWT


* EXPRESS
    Para generar un token
    ~~~~
    npm install express
    ~~~~
* JWT
    Para generar un token
    ~~~~
    npm install jsonwebtoken
    ~~~~
* COOKIE
    Para serializar un token y crear una cookie
    ~~~~
    npm install cookie
    ~~~~
* DOTENV
    Para utilizar .env
    ~~~~
    npm install dotenv
    ~~~~

* BCRYPTJS
    Para poder cifrar las contraseñas
    ~~~~
    npm install bcryptjs
    ~~~~
